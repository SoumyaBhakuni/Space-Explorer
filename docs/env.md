# Environment Variables

## PostgreSQL

- DB_HOST: database host (localhost for local dev)
- DB_PORT: database port (default 5432)
- DB_NAME: database name (space_project)
- DB_USER: database user (space_user)
- DB_PASSWORD: database password (space_pass)

## Redis

- REDIS_HOST: redis host (localhost)
- REDIS_PORT: redis port (6379)

## ML API

- ML_API_HOST: FastAPI / ML API host URL

## Feature Flags

- FEATURE_X_ENABLED: true/false
- FEATURE_Y_ENABLED: true/false
